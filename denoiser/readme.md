# Usage

## Train

```
usage: train.py [-h] [--epochs EPOCHS] [--batch_size BATCH_SIZE] [--learning_rate LEARNING_RATE]
                [--load_prev LOAD_PREV] [--test TEST] [--image_path IMAGE_PATH] [--save_img SAVE_IMG]

options:
  -h, --help            show this help message and exit
  --epochs EPOCHS
  --batch_size BATCH_SIZE
  --learning_rate LEARNING_RATE
  --load_prev LOAD_PREV
  --test TEST
  --image_path IMAGE_PATH
  --save_img SAVE_IMG
```
